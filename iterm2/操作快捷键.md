command + t：新建窗口

command + d：垂直分屏，

command + shift + d：水平分屏。

command + ] 和command + [ 在最近使用的分屏直接切换.

command + alt + 方向键：切换到指定位置的分屏。

command + 数字：切换标签页。

command + 方向键：按方向切换标签页。

shift + command + s：保存当前窗口快照。

command + alt + b：快照回放。很有意思的功能，你可以对你的操作根据时间轴进行回放。可以拖动下方的时间轴，也可以按左右方向键
