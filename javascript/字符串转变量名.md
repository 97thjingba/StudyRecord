使用eval 将字符串转换为变量名 
这个一般用于在 订阅模式下 传递消息的时候.传递给需要处理的页面的时候，不用写一大堆的判断逻辑.直接转换成变量来进行相应页面的操作

```js
this.setState({ backgroundImg:  eval(`${background}Background`) });
```

```js
 function string_to_name(string){
	 let _name = 'var new_name=' + string;
	 eval(_name);
	return _name;
}
```
